Auth Router

Handles register, login, and logout actions.

Register User

METHOD: GET

ENDPOINT: /api/auth/register/

Request

{
    "username": "demo",
    "password": "password"
}

Response

{
"id": 1,
"username": "demo",
}

Login User

METHOD: POST

ENDPOINT: /api/auth/login/

Request

    {
        "username": "demo",
        "password": "password"
    }

Response

    {
        "id": 1,
        "username": "demo",
    }

Logout User

METHOD: GET

ENDPOINT: /api/auth/logout/

Response

    {
        "message": "Logged out successfully."
    }

Photos Router

Supports full CRUD for managing photos in the database. (GET, ADD, UPDATE, DELETE)

Get Photos by User ID

METHOD: GET

ENDPOINT: /api/photos/:id

Response

            "id": 1
            "title": "Gandolf"

Add Photo

METHOD: POST

ENDPOINT: /api/photos/:id

Request

        {
            "photo_title": "Lord of the Rings Gandolf",
            "photo_url": "https://upload.wikimedia.org/wikipedia/en/e/e9/Gandalf600ppx.jpg"

        {
